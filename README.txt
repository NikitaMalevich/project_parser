Скрипт парсит сайт elibrary.ru и сохраняет данные по статьям автора в файл Excel.
Для запуска скрипта необходимо установить библиотеки и хром-драйвер.

Порядок работы:
1. Установить библиотеки 
1.1 В командной строке ввести pip install -r requierements.txt

2. Установить chromedriver
2.1 Положить в папку проекта

3. Запустить файл main.py
3.1. Ввести RINC id

4. Файл со статьями будет в папке проекта
